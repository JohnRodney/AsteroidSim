# AsteroidSim

A simulation project for modeling asteroid behavior and interactions in space.

## Description

This repository contains code for simulating asteroid dynamics, including gravitational interactions, orbital mechanics, and collision detection.

## Features

- 3D asteroid simulation
- Gravitational physics
- Collision detection
- Real-time visualization
- Configurable simulation parameters

## Getting Started

### Prerequisites

- Python 3.8+
- Required dependencies (see requirements.txt)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/JohnRodney/AsteroidSim.git
cd AsteroidSim
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the simulation:
```bash
python main.py
```
